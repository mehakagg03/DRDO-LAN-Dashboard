# DRDO-LAN-Dashboard

A lightweight, interactive dashboard built using HTML, CSS, and JavaScript to visualize internship data from CSV files designed to work offline without any server or internet connection.

## 🚀 Features ##

📂 Upload any CSV file (no backend required)

📊 Dynamic data visualization using charts

📈 Multiple chart types (Bar, Doughnut, Horizontal Bar)

🔍 Year-wise filtering

📌 KPI cards

🧠 Smart column mapping (handles different CSV formats)

📁 Supported Data Columns


## 🖥️ How to Use ##
• Download or clone this repository

• Open index.html by double-clicking (no server required)

• Upload your CSV file

• Dashboard will load automatically 🎉

## 📊 Charts Included ##

• Interns by State → Bar Chart

• Interns by Affiliation → Horizontal Bar Chart

• Interns by Mentors → Top 10 Chart

• Domain Distribution → Doughnut Chart

• Department Distribution → Bar Chart

• Duration Distribution → Pie Chart

## ⚙️ Technologies Used ##
• HTML5

• CSS3

• JavaScript (Vanilla JS)

• Chart.js

## 🧠 Key Highlights ##
• Works completely offline

• No installation or setup required

• Handles large CSV datasets

• Flexible design for different data formats

• Uses FileReader API instead of fetch for local file access

## 💼 Use Cases ##
• Internal company dashboards (secure environments)

• Academic internship tracking

• Offline data visualization

• Non-technical user reporting tools
